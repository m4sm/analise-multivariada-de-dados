# 📊 Fundamentos de Ciência de Dados, Análise Multivariada e Preparação para IA

Repositório do Grupo de Estudo em Computação Aplicada (2026/1). Este semestre focamos na transição do pensamento algorítmico puro para a análise de dados complexos, utilizando Python como ferramenta principal.

---

## 🎯 Objetivos
O objetivo central é buscar capacitação ao transformar dados brutos em conhecimento acionável, tratando conjuntos de dados como espaços vetoriais e matrizes multivariadas.

* **Domínio de Ferramentas:** Manipulação de dados com NumPy e Pandas.
* **Análise Estatística:** Aplicação de estatística descritiva e exploratória (EDA).
* **Pensamento Multivariado:** Redução de dimensionalidade (PCA) e Clustering (K-means).
* **Comunicação Científica:** Interpretação crítica de resultados e apresentações técnicas.

---

## 🛠️ Ferramentas e Organização

- **Linguagem:** Python 3.x
- **Ambiente:** Jupyter Notebooks (`.ipynb`)
- **Portfólio:** GitHub (Atualizações semanais obrigatórias)
- **Dataset:** [Breast Cancer Wisconsin (Diagnostic)](https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+(Diagnostic)) - Um dataset multivariado da área da saúde.

---

## 📅 Cronograma de Atividades

O semestre é dividido em 4 blocos de projetos práticos:

### 📦 Bloco 1: EDA e Estatística Descritiva
- **Semana 1-4:** Carregamento de dados, limpeza com Pandas, medidas de tendência central e dispersão, e visualização de dados (Histogramas, Boxplots).
- **Entregável:** Projeto 1 finalizado no Notebook.

### 📈 Bloco 2: Análise Multivariada Exploratória
- **Semana 5-8:** Correlação vs. Causalidade, Heatmaps e visualizações em N-dimensões.
- **Entregável:** Projeto 2 (Identificação de padrões e estruturas ocultas).

### 📐 Bloco 3: Redução de Dimensionalidade (PCA)
- **Semana 9-12:** Padronização de dados, Variância explicada e Projeções 2D/3D.
- **Entregável:** Projeto 3 (Simplificação de complexidade via PCA).

### 🤖 Bloco 4: Clustering e Introdução à IA
- **Semana 13-16:** Agrupamento com K-means, avaliação de modelos e a ponte conceitual com Processamento de Imagens.
- **Entregável:** Projeto 4 e Seminário Final.
